# Grid

- 浮动元素不会打乱栅格容器，即不会像块级容器那样移动到浮动元素的下方。
- 栅格容器和栅格元素外边距不折叠。
- 有些CSS属性和功能不能应用到栅格容器和栅格元素上：
  - 栅格容器上的所有column属性都被忽略；
  - 栅格容器没有::first-line和::first-letter伪元素，如果使用，将被忽略；
  - 栅格元素上的float和clear属性将被忽略。但float属性对栅格容器中的子元素的display属性的计算值有影响（栅格元素的display属性在变成栅格元素之前计算）。
  - vertical-align属性对栅格元素不起作用，不过可能影响栅格元素中的内容。
- 如果栅格容器声明为inline-grid,而目标元素是浮动的或绝对定位的，那么display的属性计算值将变为grid。

放置栅格线

栅格线始终可以使用数字引用
一条栅格线可以有多个名称，可以使用任意一个名称引用它，但不能像类名那样连在一起使用。
